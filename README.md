little-bot
==========

Installation
------------

```
npm install
```

Usage
-----

Set environment variables or create a `.env` file:

```
export TWILIO_ACCOUNT_SID=…
export TWILIO_AUTH_TOKEN=…
export TWILIO_PHONE_NUMBER_FROM=…
export TWILIO_PHONE_NUMBER_TO_1=…
export TWILIO_PHONE_NUMBER_TO_2=…
export URL_1=…
export URL_2=…
```

Then run the CLI:

```
npm start
```
