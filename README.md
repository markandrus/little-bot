little-bot
==========

Installation
------------

```
npm install
```

Usage
-----

Set environment variables or create a `.env` file:

```
export CAPTCHA_API_KEY=…
export TWILIO_ACCOUNT_SID=…
export TWILIO_ACCOUNT_SID=…
export TWILIO_PHONE_NUMBER_FROM=…
export TWILIO_PHONE_NUMBER_TO=…
export URL=…
```

Then run the CLI:

```
npm start
```
